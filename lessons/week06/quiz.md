## 🟢 第一部分：是非題 (True/False)

1.  (  ) 在 Python 中，`snake_body = []` 代表創造了一個空的變數，只能存一個數字。
2.  (  ) `snake_body.append(head)` 會把頭的位置加到清單的**最後面**。
3.  (  ) `snake_body.pop(0)` 會刪除清單中**最新**加入的元素。
4.  (  ) 如果程式沒有寫 `pop(0)`，貪食蛇的身體就會無限變長。
5.  (  ) `(100, 200)` 是一個 Tuple (元組)，它的內容不能被直接修改 (例如 `pos[0] = 50` 會報錯)。

## 💡 解答與解析 (Teachers Only)

### 是非題
1.  **False** (List 可以存多個元素)
2.  **True**
3.  **False** (pop(0) 刪除最舊/最前面的元素)
4.  **True** (這就是貪食蛇變長的原理)
5.  **True** (Tuple 是不可變的 Immutable)

### 程式碼填空
1.  `[]`
2.  `append`
3.  `pop`

### 腦力激盪
1.  `[(10, 10), (20, 10), (30, 10), (40, 10)]`
2.  `[(20, 10), (30, 10), (40, 10)]`
3.  `(40, 10)`

> **教學提示**：請學生把 List 想像成「排隊買票」或「水管裡的水珠」，新來的在後面排隊，票買完的前面離開。

# Week 02 | 隨堂測驗 (Quiz)：座標與形狀

> **測驗目標**：檢測學生是否掌握電腦座標系統 (Y 軸向下) 與 RGB 色彩混色原理。
> **測驗時間**：10 分鐘

---

## 🟢 第一部分：是非題 (True/False)

1.  (  ) 在電腦的世界裡，Y 軸座標愈大，代表位置愈往「上」移。
2.  (  ) RGB 顏色數值的範圍是從 0 到 255。
3.  (  ) `(255, 0, 0)` 是紅色。
4.  (  ) `pygame.Rect` 物件的 (x, y) 座標，是指該方塊的「正中心」。
5.  (  ) 繪製矩形時，如果設定 `width=0`，會畫出一個空心的邊框。

---

## 🔴 第三部分：腦力激盪 (Logic Challenge)

1.  **置中大挑戰**：如果視窗大小是 $800 \times 600$，我們有一個 $100 \times 100$ 的方塊，請問該方塊的「左上角」座標 (x, y) 應該設為多少，才能讓方塊剛好在畫面正中央？
    *   答案：(______, ______)

2.  **混色預測**：如果紅光 (255, 0, 0) 加上綠光 (0, 255, 0)，螢幕會顯示什麼顏色？
    *   答案：____________________

---

## 💡 解答與解析

### 是非題
1. **False** (Y 軸愈大愈往下)
2. **True**
3. **True**
4. **False** (是指左上角)
5. **False** (width=0 是填滿)

### 程式碼填空
1. `Rect`
2. `rect`
3. `width`

### 腦力激盪
1. (350, 250) -> 計算方式: $(800-100)/2=350$, $(600-100)/2=250$。
2. 黃色。

> **測驗目標**：檢測學生對類別繼承 (Inheritance)、方法覆寫 (Override) 以及多型 (Polymorphism) 應用的理解。
> **測驗時間**：10 分鐘

## 🟢 第一部分：是非題 (True/False)

1.  (  ) 類別繼承允許子類別直接擁有父類別中定義的變數與函式，不需要重新編寫。
2.  (  ) 在 Python 中，定義繼承的語法是 `class 子類別(父類別):`。
3.  (  ) 子類別中的 `__init__` 函式如果使用了 `super().__init__()`，代表它會放棄繼承父類別的初始化。
4.  (  ) 「多型」是指不同類別的物件可以擁有名稱相同的函式 (如 `draw()`)，讓主程式可以用統一的方式呼叫。
5.  (  ) 繼承的主要目的是為了讓程式碼變得更長，以便展現開發者的技術水平。

## 💡 解答與解析

### 是非題
1. **True**
2. **True**
3. **False** (剛好相反，它是為了執行父類別的初始化代碼)
4. **True**
5. **False** (是為了減少冗餘代碼，提高開發效率)

### 程式碼填空
1. `Food`
2. `super()`
3. `draw` (或 `effect`)
4. `draw`

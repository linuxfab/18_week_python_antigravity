## 💡 解答與解析

### 是非題
1. **True**
2. **False** (BGM 建議使用 `pygame.mixer.music`，它支援串流播放，不佔用大量記憶體)
3. **True**
4. **False** (`play()` 是非同步/非阻塞的)
5. **True**

### 程式碼填空
1. `load`
2. `無限循環播放`
3. `Sound`
4. `play`

### 腦力激盪
1. WAV 下載入後可即時播放、無延遲；MP3/OGG 提供高壓縮率，節省存儲空間與記憶體。
2. 開放性建議：準備 3~5 個頻率微調過的相似檔案隨機選取播放、利用 `set_volume` 做音量的微小隨機變動、或是像 Minecraft 一樣根據不同動作組合不同的音效。

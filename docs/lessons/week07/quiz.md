## 🟢 第一部分：是非題 (True/False)

1.  (  ) 在 Python 中，`def` 關鍵字是用來定義一個新的函數。
2.  (  ) 函數一旦被定義了，裡面的程式碼就會立刻自動執行。
3.  (  ) 「封裝 (Encapsulation)」的目的之一，是隱藏複雜的實作細節，讓主程式更易讀。
4.  (  ) `range(0, 10, 2)` 會產生 `0, 2, 4, 6, 8, 10` 這六個數字。
5.  (  ) 將重複的程式碼提取出來變成函數，是重構的一種常見手段。

## 🟢 第一部分：是非題 (True/False)

1.  (  ) `import random` 是使用隨機功能的必要步驟。
2.  (  ) `random.randint(0, 10)` 只會產生 0 到 9 之間的整數，不包含 10。
3.  (  ) 如果方塊移動步伐是 40，但食物生成座標是 43，這稱為「網格對齊 (Grid Alignment)」成功。
4.  (  ) 在函數內部若要修改外部定義的變數，必須使用 `global` 關鍵字宣告。
5.  (  ) 為了讓遊戲好玩，所有的變數（如視窗大小、方塊大小）都應該設成隨機的。

## 💡 解答與解析

### 是非題
1. **True**
2. **False** (randint 包含頭尾，所以會包含 10)
3. **False** (這是失敗，稱為幽靈蘋果/沒對齊)
4. **True**
5. **False** (核心規則通常是固定的，只有挑戰元素才隨機)

### 程式碼填空
1. `random` (頂部), `global` (函數內)
2. `block_size`
3. `block_size`

### 腦力激盪
1. `<= 5` (或 `>= 96`)
2. `2.5%` (1/40)

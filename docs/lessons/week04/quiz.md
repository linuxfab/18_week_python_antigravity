# Week 04 | 隨堂測驗 (Quiz)：變數與自動移動

> **測驗目標**：檢測學生對「變數更新」與「邊界反彈邏輯」的理解。
> **測驗時間**：10 分鐘

---

## 🟢 第一部分：是非題 (True/False)

1.  (  ) 在程式中，`x = x + 5` 是一個數學方程式，代表 x 等於 x 加 5。
2.  (  ) 如果要讓物體移動，我們必須在 `while` 迴圈中不斷改變它的座標變數。
3.  (  ) `speed_x = -speed_x` 代表將速度的方向取反（正變負、負變正）。
4.  (  ) `block_x < 0` 代表方塊已經撞到了視窗的右邊牆壁。
5.  (  ) 如果把 `block_x = 400` 寫在 `while` 迴圈裡面，方塊會跑不動。

---

## 🟡 第二部分：程式碼填空 (Code Completion)

請寫出具備反彈功能的簡單移動邏輯。

```python
# 更新位置
player_x ______ speed_x

# 邊界判定 (假設螢幕寬 800)
if player_x > 800 or player_x < 0:
    # 執行反彈
    speed_x = ______ speed_x
```

---

## 🔴 第三部分：腦力激盪 (Logic Challenge)

1.  **重力實驗**：如果在每一幀都執行 `speed_y = speed_y + 0.1`，方塊的運動軌跡會變成什麼樣子？
    *   答案：____________________

2.  **穿牆術**：如果不寫邊界判定，方塊會發生什麼事？
    *   答案：____________________

---

## 💡 解答與解析

### 是非題
1. **False** (它是「賦值」，將新值覆蓋舊值)
2. **True**
3. **True**
4. **False** (是左邊牆壁)
5. **True** (因為每一幀都會被重置回 400)

### 程式碼填空
1. `+=` (或 `= player_x +`)
2. `-`

### 腦力激盪
1. 會變成加速墜落的拋物線（模擬重力）。
2. 會永遠飛出畫面，消失在無盡的像素虛空中。

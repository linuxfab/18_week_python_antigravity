## 1. 環境與進度確認
* 讀取課程大綱: 檢查專案根目錄下的 `docs\syllabus.md`（或 `課程大綱.md`）。
* 確認週次: 詢問使用者目前要生成第幾週（Week X）的內容。
* 定義對象: 鎖定為「國中程度初學者」，語氣需親切、專業且詳細。

## 2. 深度教學邏輯設計 (使用 Sequential Thinking)
* 啟動工具: `sequentialthinking`
* 思考重點:
    1. 本週核心 Python 語法是什麼？(例如: if-else, for loop, or Pygame events)。
    2. 這些語法如何應用在「Snake Game」專案中？
    3. 國中生常見的邏輯痛點與「語法陷阱」有哪些？
    4. 如何設計一個具備「Vibe Coding」風格的範例（著重於視覺反饋與樂趣）？

## 3. 官方文件查閱 (備案選擇)
* 優先選用: 呼叫 `context7` 查詢本週涉及的庫（如 `pygame` 或 `random`）的最新 API。
* 備案: 若額度不足，使用 `fetch` 抓取官方文檔。

## 4. 內容產出
* 範例程式碼: 產生一個乾淨、加註詳細繁體中文註解的 `.py` 檔案。
* 學生講義: 產出一個 `.md` 檔案，包含：
    * ### 本週目標
    * ### 程式碼拆解 (詳細解說)
    * ### 動手試試看 (練習題)
    * ### 參考來源 (附上 URL)

## 5. 自動化檔案存檔 (使用 Filesystem)
* 將範例程式碼存至 `./docs/lessons/weekX/demo.py`。
* 將學生講義存至 `./docs/lessons/weekX/handout.md`。
* (選用) 將講義副本存至您的 Obsidian 指定路徑。

# 遊戲資產包 (Assets Guide)

這份資料夾存放了 Snake Game 所需的外部資源。為了符合 **Vibe Coding** 的精神，我們優先使用程式生成的資源，但您也可以根據以下指南自行擴充。

## 🎵 音效 (Sounds)
所有音效皆由 `assets/make_sounds.py` 自動生成，確保課程專案在沒有網路的情況下也能立即執行。

*   **`eat.wav`**: 蛇吃到食物時的 8-bit 高頻嗶嗶聲。
*   **`game_over.wav`**: 蛇撞牆或咬到自己時的雷射下降音 (Laser dive)。

> **進階挑戰**：您可以修改 `make_sounds.py` 中的 `freq` (頻率) 或 `duration` (長度)，創造出專屬於您的重低音音效！

## 🎨 影像 (Images)
目前遊戲主要使用 `pygame.draw` 繪製幾何圖形。若要升級為圖片版，建議規格如下：

*   **尺寸**：40x40 像素 (對應 `block_size`)。
*   **格式**：PNG (支援透明度)。
*   **建議清單**：
    *   `head.png` (蛇頭)
    *   `body.png` (蛇身)
    *   `food.png` (食物)
    *   `background.png` (背景圖，建議 800x600)
